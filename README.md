# ISCHAIN Community Website Versi 2.0

Situs ini diperuntukan untuk informasi dan pengenalan ISCHAIN Community ke khalayak ramai.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
